## Placeholder Policy

All placeholder assets are machine-generated, minimal files used solely as build targets. They contain no content derived from any copyrighted work.

## Future Asset Guidelines

When adding real assets:

1. **Original Art**: Create original sprites, tiles, and effects. Do not trace or copy existing game art.
2. **Free Assets**: If using third-party assets, only use those with permissive licenses (CC0, CC-BY, or equivalent). Attribute in this file.
3. **Fonts**: Use system fonts or freely-licensed fonts (e.g., Google Fonts with OFL license).

## Recommended Free Resources

- **Pixel Art**: [OpenGameArt.org](https://opengameart.org) — filter by CC0 license
- **Fonts**: [Google Fonts](https://fonts.google.com) — OFL licensed
- **Icons**: [Game-icons.net](https://game-icons.net) — CC-BY-SA

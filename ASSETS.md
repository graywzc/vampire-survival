## Future Asset Guidelines

When adding real assets:

1. **Original Art**: Create original sprites, tiles, and effects. Do not trace or copy existing game art.
2. **Free Assets**: If using third-party assets, only use those with permissive licenses (CC0, CC-BY, or equivalent). Attribute in this file.
3. **Fonts**: Use system fonts or freely-licensed fonts (e.g., Google Fonts with OFL license).
4. **Manifest**: Always update `assets/manifest.json` with a logical name and metadata when adding new assets.
